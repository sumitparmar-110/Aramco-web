# BuildMax Construction Landing Page

## Project name
BuildMax Construction Landing Page

## Goals
Create a modern static landing page inspired by the provided construction/manufacturing website screenshot, with a strong hero section, service highlights, featured project content, process steps, blog section, and footer contact area.

## Main features
- Full-width hero section with dark overlay and CTA buttons
- Sticky header with responsive navigation
- Floating callback/request form card in hero area
- Statistics strip below hero
- About/company section with image and experience badge
- Services grid with highlighted feature card
- Featured project section
- Work process section
- Coverage/reach section with map and metric card
- CTA banner
- Blog/news section with large and small article cards
- Footer with links, services, and contact info
- Mobile menu interaction using vanilla JavaScript

## Currently completed features
- Implemented `index.html` as the main landing page
- Added responsive styling in `css/style.css`
- Added mobile navigation behavior in `js/main.js`
- Used CDN assets for Google Fonts and Font Awesome
- Used externally hosted images to visually match the provided screenshot style
